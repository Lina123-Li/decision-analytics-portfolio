# Decision Modeling under Uncertainty
Project Overview：
1. This project analyzes voter support for the Liberal Party of Canada in the 2019 federal election using data from the 2019 Canadian Election Study (CES) web survey. The analysis focuses on how demographic factors—particularly education level and age—are associated with voting behavior.
   
2.  Using stratified sampling and statistical inference, the project estimates the expected vote share for the Liberal Party and constructs 95% confidence intervals. A logistic regression model is then developed to predict the probability of voting Liberal based on key demographic variables.
   
3. The project demonstrates how statistical modeling can be applied to understand decision behavior in real-world datasets and illustrates the role of demographic factors in shaping electoral outcomes.


# Methods
Stratified sampling estimation

Confidence interval construction

Logistic regression modeling

Model interpretation and prediction
